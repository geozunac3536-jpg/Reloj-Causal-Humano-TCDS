# Reloj Causal Humano — TCDS

Nodo experimental TCDS que usa los sensores del teléfono (acelerómetro, magnetómetro y barómetro)
para visualizar un "Reloj Causal" con:

- Aguja t_C (|a| normalizada).
- Espectro de aceleración (frecuencia pico f_a).
- Espectro magnético (frecuencia pico f_B).
- Altímetro relativo (Δh y v_z).
- Punto central modulando con el nivel de batería.
